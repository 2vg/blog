## Calpa's Blog

Source code repo of my Blog.

If you want to build a blog like this, please fork it first.

## Develop

How to start this project?

```
1. npm install
1. npm run develop
```

## Deploy

Fork this project, and run
```
npm run deploy
```

This blog is currently using Netlify.
